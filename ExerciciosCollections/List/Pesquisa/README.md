## Classe `SomaNumeros` em Java

Neste projeto, criamos uma classe chamada "SomaNumeros" que lida com uma lista de números inteiros em Java. A classe possui os seguintes métodos:

### Métodos Implementados

- `adicionarNumero(int numero)`: Este método permite adicionar um número à lista de números na instância da classe.

- `calcularSoma()`: Este método calcula a soma de todos os números presentes na lista e retorna o resultado como um valor inteiro.

- `encontrarMaiorNumero()`: Este método encontra o maior número na lista e retorna o valor desse número.

- `encontrarMenorNumero()`: Este método encontra o menor número na lista e retorna o valor desse número.

- `exibirNumeros()`: Este método retorna uma lista contendo todos os números presentes na lista.
